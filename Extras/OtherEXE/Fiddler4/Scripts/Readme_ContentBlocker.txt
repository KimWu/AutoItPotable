Prototype content-blocker allows testing of websites when some resources are blocked.

1. Install the latest Fiddler2 from http://www.fiddler2.com
2. Copy ContentBlock.dll to \Program Files\Fiddler2\Scripts or \Program Files (x86)\Fiddler2\Scripts if on a 64-bit OS
3. Run Fiddler2
4. On the "ContentBlock" menu and check the "Enabled" option

Tips:
Right-click sessions in the Session list and choose "Block this host" to block traffic from that host
Type BLOCKLIST in the QuickExec box to see the current block list
