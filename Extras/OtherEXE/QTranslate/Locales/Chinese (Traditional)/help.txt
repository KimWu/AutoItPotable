全域快速鍵 (預設值):
連點兩下 Ctrl => 顯示主視窗 (並翻譯選擇的文字)
Ctrl+Q => 在彈出視窗顯示翻譯結果
Ctrl+Alt+Q => 在網路上搜尋選擇的文字
Ctrl+E => 聽選擇文字的發音

快速鍵:
Ctrl+Enter => 在主視窗做翻譯
Ctrl+N => 清除目前的翻譯
Ctrl+H => 顯示翻譯歷史紀錄
Ctrl+K => 顯示螢幕小鍵盤
Ctrl+Tab/Ctrl+Shift+Tab/Ctrl+1..9 => 切換翻譯用的伺服器
Shift+Esc => 將翻譯語言重設為自動偵測
Ctrl+I => 翻譯語言對調
Ctrl+Space => 顯示 建議/自動完成 下拉選單
F1 => 幫助
Alt+左方向鍵 => 顯示上一個翻譯
Alt+右方向鍵 => 顯示下一個翻譯

本軟體可使用兩種翻譯模式 (了解如何使用請參考以下內容):
  1) 顯示快捷圖示 (在任何程式中選擇文字, QTranslate 的圖示會出現在選擇的文字旁. 點選圖示, 將會啟用與圖示相對應的功能);
  2) 即時翻譯 (在任何程式中選擇文字, 將會翻譯被選取的文字).
    
功能:
  - 左鍵點選工具列圖示, 啟用/關閉 滑鼠功能;
  - 左鍵雙擊工具列圖示, 開啟翻譯主視窗;
  - 左鍵雙擊彈出視窗上方, 將會切換為主翻譯視窗;
  - 左鍵點選彈出視窗下方, 可使用不同的伺服器做翻譯.
