全局快捷键（默认值）：
双击Ctrl => 显示主窗口（并显示选定文本的译文）
Ctrl+Q => 翻译选定文本并将其显示在弹出窗口
Ctrl+Alt+Q => 显示词典窗口
Ctrl+E => 听选定文本的发音

快捷键：
Ctrl+Enter => 翻译文本
Ctrl+N => 清除当前翻译
Ctrl+H => 显示翻译历史记录
Ctrl+K => 显示虚拟键盘
Ctrl+Tab/Ctrl+Shift+Tab/Ctrl+1..9 => 切换到后一个/前一个/第1~9个翻译服务
Shift+Esc => 重设语言对为自动检测
Ctrl+I => 互换翻译方向
Ctrl+空格 => 显示 建议/自动完成 下拉选单
F1 => 帮助
Alt+左箭头 => 转到前一个翻译
Alt+右箭头 => 转到后一个翻译

通过鼠标选择可使用两种翻译模式：
  1) 显示图标（在任何程序中选择文本-程序的图标会出现在光标附近。点击图标，弹出窗口里会显示选定文本的译文）；
  2) 显示译文（在任何程序中选择文本-立即弹出窗口显示选定文本的译文）。

操作：
  - 左键单击系统托盘里的程序图标以启用/关闭鼠标选择模式（见上文）；
  - 左键双击系统托盘里的程序图标显示主窗口；
  - 左键双击弹出窗口上方, 会打开包含当前翻译的主窗口；
  - 左键单击弹出窗口下方服务的名字会更改当前服务并重新翻译文本。
